<h1>Hide & Seek</h1>
<h2>About</h2>
<p><b>Hide & Seek</b> is a classic multiplayer game where players either hide in clever spots or try to find and tag the others.</p>
<h2>Features</h2>
<ul>
  <li><b>Modular & Interactive UI</b>: Includes dynamic input prompts (bottom right), round status and timer (top), an animated stamina bar (bottom left) and a chatbox (top left).</li>
  <li><b>Extensible Round System</b>: Supports multiple round stages and host-client synchronization, implemented using inheritance, generics and abstract classes.</li>
  <li><b>Gameplay Features</b>: Toggleable flashlights, player-pushing mechanics, customizable game settings, adaptive clothing for both hiders and seekers and more.</li>
</ul>
